Organic4U:A Responsive Landing-Page

Role: Frontend developer and UX/UI designer

Tool: Figma,Tailwind CSS and JavaScript

Length: 1 month

Completion Status: 100% complete

Year: 2024

Overview

Design a visually appealing, responsive website to promote organic food. Use Figma for design, Tailwind CSS for styling, and JavaScript for interactivity. Select and prioritize content to align with the organic food theme for an engaging user experience.

Designed a user-centric, responsive website using Figma, TailwindCSS, and JavaScript, while finding and prioritizing content suited for promoting organic food.

Organic Food Brands: Companies promoting their organic products and brand

Health & Wellness Bloggers: Individuals sharing organic living conten

Farmers & Markets: Local growers promoting organic produce.

Step1 Sketch: Prioritized the content, identified key information for the landing page, sketched a clean, natural layout, and sent it to mentors for review.

Step2 Prototype: Developed an interactive prototype to test user interactions.

Step3 Development:Built the website with Tailwind CSS and JavaScript. Reviewed with mentor before finalizing for Quicksite Guru.

Design Process

Step 1: Sketch

I started by prioritizing the content first and then analyzing which information was crucial for the landing page then sketched the layout to represent clean and natural vibes and sent it to mentors to review.

Step 2: Prototype

Next, I created an interactive prototype to visualize and test user interactions before development, ensuring a smooth user experience.

Step 3: Development

I used Tailwind CSS and JavaScript to develop the website, incorporating feedback from my mentor. After the final review, the website was ready for Quicksite Guru to implement.
